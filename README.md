## Overview

Event Explorer is a React Native mini app developed as part of the College Crave App Developer Intern coding round. The app showcases core skills in authentication UI, API integration, event data display, and navigation, demonstrating practical app development within a limited timeframe.

---

## Features Implemented

- **Login Screen:** User-friendly authentication interface with field validation.
- **Signup Screen:** Allows users to enter credentials and navigate to login.
- **Event List Screen:** Fetches and displays event data from a live public API (GitHub Gist).
- **API Integration:** Real-time HTTP fetch of event data with loading indicators and error handling.
- **Navigation:** Seamless screen transitions using React Navigation.
- **(Optional) Event Saving:** Single-user event saving via AsyncStorage (if implemented).

## Setup Instructions

1. Clone the repository:

git clone https://github.com/kushagrakartikeye/college-crave.git
cd college-crave

text

2. Install dependencies:

npm install

text

3. Start Metro Bundler:

npx react-native start

text

4. Run the app on Android emulator or device:

npx react-native run-android

text

> Make sure your emulator or device is running and connected to the same network as your development machine for the API fetch to work.
